# Índice Geral de Agentes Evolutivos no Formato Juniper

Este índice lista e descreve os prompts de instrução e os documentos de referência conceitual para os três Agentes Evolutivos criados, todos seguindo a metodologia **Juniper** (Markdown, Python, Multilinguismo) e o princípio de **Evolução** através da documentação estruturada.

## 1. Agente Interpretador de Juniper (Foco: Conversão de Dados e Documentação)

Este agente é o especialista em transformar qualquer arquivo (texto, código) no formato estruturado Juniper, utilizando Python para a implementação e Markdown para a documentação que serve como dado de treinamento.

| Documento | Descrição |
| :--- | :--- |
| `prompt_instrucao_revisado.md` | Prompt de sistema detalhado para o Agente Interpretador de Juniper. |
| `documento_referencia_conceitual_revisado.md` | Documento que define o Formato Juniper Estruturado e o Mecanismo Evolutivo. |

## 2. Agente Especialista em Infraestrutura (Foco: Pastas, Versionamento e Cloud)

Este agente é o especialista em DevOps, definindo e documentando as melhores práticas para Gerenciamento de Pastas, Versionamento (Git/GitHub) e Integração Cloud (Infraestrutura Google).

| Documento | Descrição |
| :--- | :--- |
| `prompt_infra_evolutivo.md` | Prompt de sistema detalhado para o Agente Especialista em Infraestrutura. |
| `doc_infra_evolutivo.md` | Documento que define os pilares de expertise em Infraestrutura e o Mecanismo Evolutivo de Segurança. |

## 3. Agente Especialista em Compatibilidade de OS (Foco: Linux, Windows e Mac)

Este agente é o especialista em portabilidade, garantindo que o código e a infraestrutura funcionem de forma consistente nos sistemas operacionais Linux, Windows e macOS.

| Documento | Descrição |
| :--- | :--- |
| `prompt_os_evolutivo.md` | Prompt de sistema detalhado para o Agente Especialista em Compatibilidade de OS. |
| `doc_os_evolutivo.md` | Documento que define os pilares de expertise em Compatibilidade de OS e o Mecanismo Evolutivo de Portabilidade. |
